## üöÄ System Features
- **Dual-Factor Security:** Face Recognition + 4-digit PIN fallback.
- **Hardware Triggered:** System activation via physical button press (GPIO 22).
- **Security Logic:** - 3 failed attempts lead to a 15-second lockout.
    - Silent mode toggle for discreet operations.
- **Edge AI:** Real-time processing using the **IMX500 AI Camera**.
